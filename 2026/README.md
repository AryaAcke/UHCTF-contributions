# 2026 Edition

A few years ago, I came up with the idea of using the numbers of pi as encryption for a CTF challenge. This was long before the LLM boom, so it would've been a very fun medium-difficulty challenge when it clicks that the OTP was the numbers. Unfortunately since then with ChatGPT, Claude and all the other new technologies, this challenge becomes trivial.

Nevertheless I made the challenge half a year before the CTF, just to already get something ready. It's a fun little challenge now and I'm happy I made it. I ended up having negative time for the rest of the year, so did not create anything else...
